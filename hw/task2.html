Lesson 15 - Homework - Task 2

  1. Проверка диапазона
    Пользователь вводит два числа: первое — начальное значение диапазона, второе — конечное. Затем вводит третье число для проверки. Программа:
    • Проверяет, находится ли число в пределах диапазона с помощью isFinite().
    • Указывает, является ли число целым с помощью Number.isNaN() и parseInt().
    • Выводит разницу между концами диапазона с помощью Math.abs().
