Lesson 15 - Homework - Task 5

  1. Анализ строки
    Пользователь вводит строку. Напишите программу, которая:
    • Определяет длину строки с помощью length.
    • Преобразует строку в верхний и нижний регистры.
    • Проверяет, содержит ли строка подстроку JavaScript с помощью includes().
    • Заменяет все пробелы на дефисы с помощью replaceAll().
